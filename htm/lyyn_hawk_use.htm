LYYN Hawk Operating Tips

Early models of the Pro 4 control panel included a membrane keypad to control the LYYN functions. VideoRay Cockpit software can now be used to control the LYYN functions. See the LYYN documentation for information about using the keypad and the section in the VideoRay Cockpit Guide for more information about the software controls.

Hardware Set Up

The LYYN is factory installed in the control panel and does not require any hardware set up.

Operation

The LYYN keypad (physical or onscreen) includes an On/Off button to turn the LYYN on or off. The amount of LYYNification can be controlled by pressing the increase or decrease buttons. The size and location of the LYYNified area can be controlled by pressing the size button.

This equipment is optional and may not be included in all VideoRay Pro 4 configurations.
